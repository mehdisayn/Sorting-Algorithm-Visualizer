Welcome to the Sorting Algorithm Visualizer

Watch sorting and searching algorithms run step by step. A quick tour:

  • Pick
    Choose an algorithm

    10 sorts and 2 searches in the Algorithm dropdown.

  • Play
    Run, Pause, Step

    Play it, pause anytime, or step through one operation at a time.

  • Data
    Shape the input

    Set the Size, pick a distribution, or enter Custom values (Space adds commas).

  • Load
    Upload or drag-and-drop

    Drop a .txt, .csv or .json file of numbers anywhere on the page.

  • Learn
    How it works

    Click the algorithm name or the ? button for the idea, complexity, and pseudocode.

  • Hear
    Sound on by default

    Each step plays a tone pitched to its value. Toggle with the ♪ button or M.

  • Save
    Export results

    Download a run report (.txt) or a visualization (.png) after a run.
